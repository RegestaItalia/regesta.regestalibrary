(static) checkParameters(functionName, parameters)

Source:
  • helper/JsHelper.js, line 26

Checks an array of mandatory parameters and throws an exception if a specified parameter is initial or has wrong type.

Parameters:
Name Type Description
functionName string

The name of the function from which this is called (for logging purposes).
parameters array

An array of parameters to check. Each parameter have to be structured as follows:

Properties
Name Type Description
name string

The name of the parameter.
value *

The value of the parameter.
expected array

An array of possible types (as strings) of the parameter.
Throws:

Will throw an error if one of the parameters is blank or of the wrong type.
